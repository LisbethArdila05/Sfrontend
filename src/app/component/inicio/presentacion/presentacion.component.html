Sistema para viveros

  • Dashboard

    Presenta la información más reciente sobre las variables ambientales de las plantas en un vistazo.

  • Lectura de variables

    Utiliza dos sensores para medir las condiciones ambientales de las plantas, como humedad y temperatura, para verificar su estado actual.

  • Analisis

    Permite al viverista evaluar el estado de las plantas mediante gráficas y datos obtenidos, proporcionando una comprensión clara de su entorno de crecimiento..

  • Historial

    Muestra un registro de todas las lecturas anteriores, con gráficas que resumen el promedio de las variables ambientales a lo largo del tiempo, facilitando el seguimiento de tendencias.
